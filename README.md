# Retail-Inventory-Excel-Project
Excel-based Retail inventory and sales analysis project

## Tools Used
- Microsoft Excel

## Project Description
This project focuses on analyzing retail inventory and sales data using Microsoft Excel. 
The dataset includes product details, store information, inventory levels, and sales transactions.

## Sheets Included
- Product_Master: Product category, supplier, cost & selling price
- Store_Master: Store and location details
- Inventory_Data: Opening stock, closing stock, reorder level
- Sales_Data: Date-wise sales quantity and revenue

## Key Analysis
- Inventory level monitoring
- Sales performance analysis
- Category-wise and store-wise insights
- Identification of low-stock products

## Use Case
This Excel project helps retail managers make data-driven decisions for inventory planning and sales optimization.
